En el caso de variables cuantitativas ademas si los datos son muy variados, que para se clasificados adecuadamente, necesitan generarse particiones de longitudes semejantes entonces se utiliza el siguiente proceso; el número de las particiones rr se consideran de acuerdo a tres criterios

  • Criterio del investigador rr no puede ser más de 20 ni menos de 5

  • r=nr=\sqrt{n} donde nn es el número de datos

  • La regla de Starges que consiste en considerar la fórmula r=3.322⋅log10nr=3.322\cdot\log_{10} n Una vez establecido el número de particiones se procede a generar los límites laterales de cada una de las particiones, sea LL la longitud de todo el conjunto es decir L=xmax−xminL=x_{\text{max}}-x_{\text{min}} entonces la longitud de las particiones o amplitud interválica se obtiene con l=Lrl=\frac{L}{r}

Tenga en cuenta que nn es el número de datos, es decir n=f1+f2+…+fr=∑i=1rn=f_1+f_2+\ldots+f_r=\sum_{i=1}^r donde fif_i es número de datos en la partición XiX_i, una de las rr particiones del conjunto total de datos.

  • Las frecuencias absolutas fif_i indican el número de datos con la característica XiX_i.
  • Las frecuencias absolutas acumuladas menor que FiF_i obedecen a la fórmula Fm=f1+f2+…+fm=∑i=1mfiF_m=f_1+f_2+\ldots+f_m=\sum_{i=1}^mf_i
  • Las frecuencias absolutas acumuladas mayor que Fi*F_i^* obedecen a la fórmula Fm*=fm+fm+1+…+fr=∑i=mrfi=n−∑i=1m−1fi=n−(f1+f2+…+fm−1)F_m^*=f_m+f_{m+1}+\ldots+f_r=\sum_{i=m}^rf_i=n-\sum_{i=1}^{m-1}f_i=n-\left(f_1+f_{2}+\ldots+f_{m-1}\right)
  • Las frecuencias absolutas relativas obedecen a la fórmula hm=fmnh_m=\frac{f_m}{n}
  • Las frecuencias absolutas relativas menor que obedecen a la fórmula Hm=fmnH_m=\frac{f_m}{n}

  • Las frecuencias absolutas relativas mayor que obedecen a la fórmula Hm*=FmnH_m^*=\frac{F_m}{n}

  • Las frecuencias absolutas relativas porcentuales obedecen a la fórmula hi%=100⋅hih_i\%=100\cdot h_i
  • Las frecuencias absolutas relativas menor que porcentuales obedecen a la fórmula Hi%=100⋅HiH_i\%=100\cdot H_i

  • Las frecuencias absolutas relativas mayor que porcentuales obedecen a la fórmula Hi*%=100⋅Hi*H_i^*\%=100\cdot H_i^*
